# FinTrack 

FinTrack é um sistema de controle de finanças pessoais desenvolvido em Java (versão console).

## Funcionalidades

- Adicionar transações (receitas e despesas)
- Listar todas as transações
- Mostrar saldo atual
- Remover transações

## Tecnoloogias utilizadas

- Java
- Progrtramação Orientada a Objetos (POO)
- ArrayList
- Tratamento de esceções
- Estrutura em pacotes
